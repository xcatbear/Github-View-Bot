# GitHub View Bot
Bot for Profile Views on the Github!

* ## Installation
```
git clone https://github.com/xcatbear/Github-Views-Botter
```
```
cd Github-View-Bot
```
```
pip install -r requirements.txt
```
```
views.py
```
